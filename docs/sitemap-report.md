## 기술 스펙

- **캐시**: `revalidate = 3600` (ISR, 1시간마다 백그라운드 갱신)

- **XML 이스케이프**: `&`, `<`, `>`, `"`, `'` 전부 처리
- **매치 쿼리**: DB RPC 함수 `get_sitemap_matches(league_id)` 사용 (JSON 파싱 서버측 처리)
- **미들웨어**: `sitemap[^/]*\.xml` 패턴으로 전체 제외
